# Ascii register

## Setup

### Requirements

To compile the website:
- `java`

To run the server:
- `nodejs`
- `npm`

### First start

```bash
cd web
npm install
```

### Start server

```bash
node web/index.js
```

### Compile website

```bash
./gradlew build
```
